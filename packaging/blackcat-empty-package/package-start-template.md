# 启动黑猫空白版

这份包是空白版：

- 没有默认账号。
- 没有预置数据库内容。
- 不会复用你电脑上别的黑猫数据库。
- 包里已经带了 Java，不需要另外装 Node 才能打开前端页面。

## 使用步骤

1. 确认 Docker Desktop 已经安装好，并且当前是运行状态。
2. 直接双击 `launch-blackcat.cmd`。
3. 第一次启动会自动准备独立的 MySQL 容器，并下载 `mysql:8.1`，这一步可能需要几分钟。
4. 启动完成后，浏览器会自动打开黑猫登录页。
5. 如果浏览器没有自动打开，手动访问：`http://127.0.0.1:18880/login`

## 第一次登录怎么做

这份包没有默认账号。第一次打开时，先进入初始化管理员页面，再创建你自己的账号和密码。

初始化要用到下面这个重置密钥：

`{{RESET_KEY}}`

填完重置密钥、用户名和新密码后，提交即可。之后就用你自己设置的账号密码登录。

## 这份空白包和旧数据的关系

- 这份包会用独立的 MySQL 容器：`{{CONTAINER_NAME}}`
- 它使用独立的数据卷，不会直接碰你机器上别的黑猫库。
- 这份包内部数据库端口是：`{{DATABASE_PORT}}`

## 常见问题

### 双击启动后没有反应

先看 Docker Desktop 是否真的已经启动完成。Docker 还没起来时，黑猫不会正常打开。

### 第一次启动很慢

第一次要下载 MySQL 镜像，这是正常现象。网络慢时会更久，先等一会儿。

### 页面打不开

直接在浏览器里输入：`http://127.0.0.1:18880/login`

### 已经初始化过一次，还想重新变回空白

这份包的数据已经写进 Docker 里的独立数据卷了。要回到全新状态，删掉对应的容器和数据卷后再启动。
