# 自动做账 v1.1.3 安装和使用说明

## 启动

双击程序目录里的 `launch-blackcat.cmd`，或桌面上的「自动做账启动」快捷方式。

默认打开：

```text
http://127.0.0.1:18880/bookkeeping
```

## WhatsApp 登录

进入「WhatsApp 群配置」。

- 如果 WhatsApp 已连接，页面会显示已连接。
- 如果还没登录，页面会显示扫码二维码。
- 手机 WhatsApp 扫码后，点击「同步本机群聊」。

## Telegram 登录

进入「Telegram 群配置」。

先填写 `API_ID` 和 `API_HASH`，保存后按页面提示扫码登录，再点击「同步本机群聊」。

## 系统更新

进入「系统更新」，点击检查更新。发现新版后点击升级，程序会自动下载更新包、覆盖程序文件并重启。

更新不会覆盖本地账号、群配置、WhatsApp 登录会话、Telegram 登录会话和生成的报表。

## 本版修复

- WhatsApp 读取服务会随主程序自动启动。
- WhatsApp 登录状态会自动刷新。
- 未登录时会显示 WhatsApp 扫码入口。
- 更新包兼容旧版系统更新功能。
