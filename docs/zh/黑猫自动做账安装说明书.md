# 黑猫自动做账安装说明书

适用版本：v1.0.8

## 一、安装前准备

使用电脑需要满足：

- Windows 10 或 Windows 11。
- 能正常联网。
- 已安装 Docker Desktop。
- 解压工具可以正常解压 `.zip` 文件。

如果电脑还没有 Docker Desktop，先看安装包里的 `01-install-docker-desktop.md`。

## 二、下载哪个安装包

新电脑第一次安装，下载：

- `AutoBookkeeping-Blank-v1.0.8.zip`

已有旧版本，只更新程序，下载：

- `auto-bookkeeping-update-v1.0.8.zip`

## 三、新电脑安装步骤

1. 解压 `AutoBookkeeping-Blank-v1.0.8.zip`。
2. 打开 Docker Desktop，等它显示已经运行。
3. 双击解压目录里的 `launch-blackcat.cmd`。
4. 第一次启动会自动准备数据库，时间可能会比较久。
5. 浏览器会自动打开：`http://127.0.0.1:18880/bookkeeping`
6. 本机使用不需要输入账号密码，会自动免密进入。

以后只想重新打开前端页面，双击：

- `open-blackcat-frontend.cmd`

如果程序没有运行，仍然需要先双击 `launch-blackcat.cmd`。

## 四、旧版本更新步骤

1. 下载 `auto-bookkeeping-update-v1.0.8.zip`。
2. 在旧版本页面里使用系统更新功能，或按旧版本已有的更新流程选择这个更新包。
3. 更新包只覆盖程序文件，不会覆盖业务数据、配置、日志、备份和本地数据目录。
4. 更新完成后重新打开 `launch-blackcat.cmd`。

## 五、常见问题

### 打开后还是进了登录页

先确认打开的是 v1.0.8 的包，并且地址是：

`http://127.0.0.1:18880/bookkeeping`

不要手动打开 `/login`。

### 页面打不开

先确认 Docker Desktop 已经运行，再重新双击 `launch-blackcat.cmd`。

### 第一次启动很慢

第一次需要下载 MySQL 镜像并创建本地数据库，等待几分钟是正常的。

### 想重新变成全新空白数据

停止程序后，删除对应的 Docker 容器和数据卷，再重新启动空白包。

### 只想打开前端

双击 `open-blackcat-frontend.cmd`。如果后台服务没有启动，这个快捷入口只能打开页面地址，不能代替完整启动。
